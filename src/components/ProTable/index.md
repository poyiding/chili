# ProTable 高级表格

项目中我们经常会写很多查询table的页面，针对这些样板代码，我们推荐采用[ProTable](https://procomponents.ant.design/components/table)。

## 查询表格

```jsx | inline

import React from 'react';

export default () => {

  return (
    <div>
      <iframe 
        width="100%"
        height='500px'
        style={{ border: 'none' }}
        src='https://procomponents.ant.design/~demos/table-single'
      />
    </div>
  );
};
```
